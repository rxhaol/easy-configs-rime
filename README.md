# 自定义主题

对于亮色主题，我本人比较喜欢的设计原则遵循：

- 背景色：`#FFFFFF`
- 候选字颜色： `#454545`
- 标签（未选中）颜色： `#8E8E8E`
- 候选字（选中）文字颜色：根据个人喜好选择
- 候选字（选中）背景颜色：`候选字（选中）文字`\*69%透明度（Alpha 的值为 80/0x50）
- 标签（选中）颜色：和候选字（选中）文字颜色保持一致

颜色参考：[中国传统颜色](https://www.zhongguose.com)
